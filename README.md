# trell

> Easy generate release ticket on Trello based on your git commits

## How it works

**Configuration options**:

- Repo path
- Board name
- List name
- Label color on card
- Adding all board members to the card by default

## Install

```bash

npm i -g trell

```

## Usage
  
```bash

trell

```
